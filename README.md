**Project Overview**

The Tiffin Subscription Web Application is a web-based solution designed to simplify meal subscription management.
It allows users to explore meal plans, subscribe based on their preferences, and track their subscriptions through a user-friendly interface. 
The project focuses on clarity, usability, and smooth user flow.

**Tech Stack**

Frontend: HTML, CSS 
Backend: Python (Flask) 
Database: MySQL

**Key Features**
-User registration and login
-Flexibility to customize meal plan selection (trial / weekly / monthly/quaterly)
-Ability to skip meals for specific days with an active subscription
-Option to order extra meals under an active subscription plan
-Subscription form with proper input validations
-Prevention of invalid inputs such as past dates
-Subscription history tracking
-Clean and structured user interface

This project is developed as an academic and learning project to demonstrate understanding of web application flow, form validation, and user-focused design.
